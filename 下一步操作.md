## 快速开始（3 步）

### 第 1 步：配置 Supabase（5 分钟）

1. **创建 Supabase 项目**
   - 访问 https://supabase.com
   - 登录/注册
   - 创建新项目
   - 等待初始化完成

2. **获取 API 密钥**
   - Dashboard > Settings > API
   - 复制 **Project URL** 和 **anon public** key

3. **设置数据库**
   - Dashboard > SQL Editor
   - 新建查询
   - 复制 `supabase-setup.sql` 的全部内容
   - 粘贴并运行

### 第 2 步：创建环境变量文件（2 分钟）

在项目根目录创建 `.env.local` 文件：

```bash
# 方法 1: 使用脚本（推荐）
./setup-env.sh

# 方法 2: 手动创建
touch .env.local
```

然后编辑 `.env.local`，填入：

```env
NEXT_PUBLIC_SUPABASE_URL=https://你的项目.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=你的anon_key
```

### 第 3 步：启动开发服务器

```bash
npm run dev
```

打开浏览器访问：http://localhost:3000

## 如果遇到问题

### 问题：Supabase 连接错误
- 检查 `.env.local` 中的 URL 和 Key 是否正确
- 确认 Supabase 项目已激活
- 检查数据库表是否已创建

### 问题：端口 3000 被占用
```bash
npm run dev -- -p 3001
```

### 问题：样式不显示
- 确认 Tailwind CSS 已安装（已在 node_modules 中）
- 重启开发服务器

### 问题：数据库操作失败
- 确认已运行 `supabase-setup.sql`
- 在 Supabase Dashboard > Table Editor 中检查 `choices` 表是否存在
