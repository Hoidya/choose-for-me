## 部署准备

### 需要做的事情

1. **推送代码到 GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/choose-for-me.git
   git push -u origin main
   ```

2. **更新 Supabase 数据库**
   - 运行 `supabase-setup.sql`（已包含 coin 类型）

3. **部署到 Cloudflare Pages**
   - 按照 `cloudflare-pages-deploy.md` 指南操作
   - 设置环境变量
   - 等待部署完成
