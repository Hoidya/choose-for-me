## 🚀 使用方法

### 启动开发服务器
```bash
npm run dev
```

### 使用 Localtunnel 分享
```bash
# 方法 1: 使用 npx（推荐）
npx localtunnel --port 3000 --subdomain choose-for-me

# 方法 2: 使用 npm 脚本（需要先安装 localtunnel）
npm install -g localtunnel
npm run tunnel
```

### 访问应用
- 本地: http://localhost:3000
- 公网: https://choose-for-me.loca.lt (使用 Localtunnel 后)

## 📝 数据库更新

如果之前已经创建了数据库，需要更新 `choices` 表：

```sql
ALTER TABLE choices 
DROP CONSTRAINT IF EXISTS choices_type_check;

ALTER TABLE choices 
ADD CONSTRAINT choices_type_check 
CHECK (type IN ('wheel', 'dice', 'card', 'coin'));
```

或者重新运行 `supabase-setup.sql`（已更新支持 coin 类型）。
