# 🔧 白色页面修复步骤

## 快速修复（按顺序尝试）

### 步骤 1: 清理缓存并重启

```bash
# 停止开发服务器 (Ctrl+C)

# 清理 Next.js 缓存
rm -rf .next

# 重新启动
npm run dev
```

### 步骤 2: 检查浏览器

1. **硬刷新页面**
   - Mac: `Cmd + Shift + R`
   - Windows: `Ctrl + Shift + R`

2. **清除浏览器缓存**
   - 打开开发者工具 (F12)
   - 右键点击刷新按钮
   - 选择"清空缓存并硬性重新加载"

3. **检查控制台**
   - 打开开发者工具 (F12)
   - 查看 Console 标签
   - 查看是否有红色错误信息

### 步骤 3: 测试简单页面

访问测试页面：http://localhost:3000/test

如果测试页面能显示，说明 Next.js 正常工作，问题在首页组件。

### 步骤 4: 检查终端错误

查看运行 `npm run dev` 的终端窗口，是否有：
- 编译错误
- 警告信息
- 模块加载错误

## 常见问题

### 问题 1: 控制台显示 "Cannot find module"

**解决：**
```bash
rm -rf node_modules package-lock.json .next
npm install
npm run dev
```

### 问题 2: CSS 未加载

**检查：**
- 浏览器 Network 标签
- 查看 `/_next/static/css/` 文件是否加载成功

**解决：**
```bash
rm -rf .next
npm run dev
```

### 问题 3: JavaScript 错误

查看浏览器控制台的错误信息，常见错误：
- `ReferenceError` - 变量未定义
- `TypeError` - 类型错误
- `Module not found` - 模块未找到
